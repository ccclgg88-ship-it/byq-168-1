## 1. 产品概述

极简AI测评入口页——核心承诺「零门槛」：拖一张头像、敲一个昵称，即可开始AI人格测评。无需注册、无需填问卷，3秒内完成输入，生成过程中以趣味标签云动效留住用户。

- 目标用户：希望快速获取AI人格画像的年轻用户群体
- 核心价值：极致极简的测评启动体验，让等待也有趣

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客 | 无需注册 | 上传头像、输入昵称、启动测评 |

### 2.2 功能模块

1. **测评入口页**：头像上传区、昵称输入框、开始测评按钮、标签云加载态、错误提示

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 测评入口页 | 头像上传区 | 支持拖拽/点击上传，圆形裁切预览，超2MB提示压缩 |
| 测评入口页 | 昵称输入框 | 2-12字符限制，空值时按钮置灰 |
| 测评入口页 | 开始测评按钮 | 非空昵称+有效头像时可点击，点击后进入加载态 |
| 测评入口页 | 标签云加载态 | 屏幕中央随机飘动趣味标签，作为等待动画 |
| 测评入口页 | 错误提示 | 非图片格式/读取失败时Toast提示 |

## 3. 核心流程

用户打开页面 → 拖拽/点击上传头像（圆形预览） → 输入昵称（2-12字符） → 点击「开始测评」 → 全屏标签云加载动画 → 等待AI结果返回

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深紫黑底(#0d0b1a) + 霓虹青色(#00f5d4) + 柔粉(#f72585)点缀
- **按钮风格**：圆角胶囊按钮，霓虹渐变填充，悬停发光
- **字体**：标题使用 ZCOOL KuaiLe（趣味中文），正文使用 Noto Sans SC
- **布局风格**：垂直居中单列布局，大量留白，视觉聚焦中央
- **图标风格**：线条感轻量图标，配合微动画

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 测评入口页 | 头像上传区 | 虚线圆形拖拽区，拖入时边框发光，上传后显示圆形头像预览，覆盖编辑图标 |
| 测评入口页 | 昵称输入框 | 底部线条输入框，居中对齐，占位文字浅灰，聚焦时线条变霓虹色 |
| 测评入口页 | 开始测评按钮 | 胶囊按钮，渐变霓虹青→柔粉，空状态灰显，hover发光+微上浮 |
| 测评入口页 | 标签云加载态 | 全屏半透明深色遮罩，中央标签随机飘动、缩放、旋转，不同颜色不同速度 |
| 测评入口页 | 错误提示 | 顶部Toast，红色背景，白色文字，3秒自动消失 |

### 4.3 响应式

- 桌面优先设计，移动端自适应
- 上传区域和输入框在移动端适当缩小
- 标签云动画在所有设备上全屏展示
